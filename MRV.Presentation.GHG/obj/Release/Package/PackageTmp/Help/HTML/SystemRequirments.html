System Requirments

Introduction ››

EasyPay’s recommended system and hardware requirments include:


  • Windows XP, Windows Vista, Windows 7 or Windows 8
  • 512MB of RAM
  • 250MB of free disk space
  • 1024x768 screen resolution or higher
  • IIS Server
  • SQL Server 2005, SQL Server 2008 or SQL Server 2012
